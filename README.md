# ToDoList
## Overview :
- By using this program you can arrange you tasks with some operations.
## Usage :
- struct 
- enum
- class 
The functins that I used is :
- Add -> func add(newItem: List)
- Create -> func create(newList: [List])
- Remove -> func remove(remove: Int)
- Update -> func update (index : Int , update : List)
- GetAllItems -> func getAllItems()
- CheckPriority -> func checkPriority(priority: priority)
- CheckStatus -> func checkStatus (status: status)
